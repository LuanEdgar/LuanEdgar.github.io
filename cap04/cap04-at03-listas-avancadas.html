Glossário de Termos Técnicos

HTML
Linguagem de marcação usada para criar páginas web.
CSS
Linguagem usada para estilizar páginas web.
JavaScript
Linguagem usada para adicionar interação às páginas.
DOM
Estrutura que representa os elementos da página.
Permite manipular conteúdos com JavaScript.
API
Interface usada para comunicação entre sistemas.
Pode fornecer dados para aplicações web.
HTTP
HTTPS
Protocolos usados para comunicação na internet.
URL
Endereço Web
Localização de um recurso na internet.
DNS
Sistema que traduz nomes de sites em endereços IP.
